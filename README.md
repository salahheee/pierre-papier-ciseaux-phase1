Pierre Papier Ciseaux – Phase 2
Description

La Phase 2 du projet Pierre-Papier-Ciseaux permet maintenant de jouer une manche complète contre l’ordinateur avec une interface graphique Tkinter.
Le jeu récupère le choix de l’utilisateur, génère un choix aléatoire pour l’ordinateur, détermine le gagnant et affiche le résultat à l’écran.

Fonctionnalités

Fonction play() : gère la logique du jeu et détermine le gagnant.

Fonction Reset() : réinitialise la saisie et le résultat pour recommencer une manche.

Fonction Exit() : ferme l’application proprement.

Affichage clair du choix de l’utilisateur, du choix de l’ordinateur et du résultat de la manche.
